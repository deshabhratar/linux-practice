# Linux Log Analyser

## What is this project?
This project analyses a Linux log file to extract meaningful
information such as errors, warnings, and recent issues.

## What does it do?
- Calculates log file size
- Counts total log entries
- Extracts ERROR, WARNING, and INFO logs
- Shows most recent ERROR entries

## Tools used
grep, wc, du, tac, bash

## How to run
Execute the script:
./log_analyser.sh
